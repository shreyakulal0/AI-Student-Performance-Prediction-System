🎓 AI Student Performance Prediction System

📌 Project Overview

The **AI Student Performance Prediction System** is a Machine Learning based analytics project designed to predict student academic performance using key educational indicators.

The system evaluates multiple regression models and automatically selects the best performing model based on R² score. It then generates performance insights such as class average, topper identification, weak student detection, grade classification and personalized academic improvement suggestions.

This project demonstrates practical implementation of **Predictive Analytics, Model Evaluation and Data Visualization** techniques.

🚀 Key Features

* Automatic dataset loading (CSV / Excel)
* Machine Learning model comparison and selection
* Student marks prediction
* Grade classification system
* Class analytics (Average, Topper, Weak Students)
* Feature importance visualization (Random Forest)
* Interactive student analytics dashboard using dropdown selection
* Personalized performance improvement suggestions
* Prediction accuracy visualization

🤖 Machine Learning Models Used

* Linear Regression
* Decision Tree Regressor
* Random Forest Regressor

The system automatically selects the best model using **R² Score evaluation**.

📊 Input Features

* Study Hours
* Attendance Percentage
* Assignment Score
* Internal Marks

Target Variable:
* Final Marks
